# 요구사항 분석

## 서비스 개요
- **목적**: 사용자가 입력한 주소를 기준으로 가까운 약국 3곳을 추천
- **데이터**: 약국 현황 데이터 (위도, 경도 포함)

## 입력
- **주소 입력 형식**: 도로명 주소 또는 지번 주소
  - 예시: 서울 중랑구 ~~~
- **주소 정확성**: 정확한 상세 주소(동, 호수)를 제외한 주소 정보 사용

## 주소 입력 방법
- **우편번호 서비스 사용**: Kakao 우편번호 서비스 이용하여 정확한 주소 입력

## 처리 과정
1. **주소 변환**: 입력 받은 주소를 위도와 경도로 변환
2. **거리 계산**: Haversine formula를 이용하여 두 위도, 경도 좌표 사이의 거리 계산
3. **약국 필터링**: 입력한 주소 정보에서 반경 10km 내에 있는 약국 추천
   - 지구는 완전한 구형이 아니므로 약간의 오차 발생 가능

## 결과
- **추천 약국**: 가까운 약국 3곳 추출
- **추출 데이터 제공 방식**:
  - 길안내 URL (shorten url로 제공)
    - base62를 통한 인코딩
    - shorten url의 유효 기간은 30일
  - 로드뷰 URL

## Haversine Formula
- 두 점 사이의 최단 거리를 구하기 위해 사용

## 기타 사항
- 추천 반경: 10km
- 서비스 데이터: 약국의 위치 정보 (위도, 경도 포함)
